Ordner​

dist - Enthält die fertigen builds. Mehr dazu unter: Build erstellen.
electron - Enthält Electron Dateien (Back-End). Mehr dazu unter: Electron.
node_modules - Enthält die installierten Node Modules.
powershell - Enthält die PowerShell Skripte, die die Daten des PCs erheben. Diese Daten werden in der Electron App dargestellt. Mehr dazu unter: Powershell-Erweiterbarkeit.
public - Enthält die Views (Front-End). Diese Dateien werden automatisch von Svelte generiert.
src - Enthält die Svelte Dateien. In diesen kann man das Front-End anpassen. Mehr dazu unter: Svelte.

Dateien​

.gitignore - Hier könnte man Ausnahmen definieren, wenn man Git benutzen will.
package-lock.json - Wird automatisch von NPM generiert.
package.json - Projektinformationen und dependencies. Mehr dazu unter package.json.
rollup.config.js - Rollup-Config (Kommt von Svelte).
